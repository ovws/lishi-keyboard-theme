# 李氏三拼3x5a键盘 - 双字体版

李氏三拼3x5a键盘主题的自定义字体版本。

## 字体配置

- **中文**：HYHuaGuan_55W.ttf（汉仪华冠55W字体）
- **英文**：TX-02-Regular.otf（TX-02 Regular字体）
- **备用**：PingFangSC-Regular（苹方常规体）

## 支持的键盘布局

- 26键拼音（横屏/竖屏，带反查版本）
- 9键拼音宫格（竖屏，带反查版本）
- 李氏双拼（lssp/lsspwsd）
- 英文26键
- 数字9键
- 符号键盘
- 表情键盘
- 浮动窗口（搜索、配置、切换）

## 主题特色

- 深色/浅色主题
- 自定义按键音效
- 优化的字体显示效果

## 安装说明

下载 `李氏三拼3x5a键盘_双字体版.zip` 并导入到你的输入法应用中。
