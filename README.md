李氏三拼3x5a键盘主题包，提供三个版本。

## 下载

### 版本 1：微调版（推荐）⭐

📦 [李氏三拼3x5a键盘_微调版.hskin](https://github.com/ovws/lishi-keyboard-theme/raw/main/李氏三拼3x5a键盘_微调版.hskin) (966KB)

**特点：**
- ✅ **李氏三拼主字体缩小 85%** - 按键上的主要字符不再过大

## 使用建议

### 选择微调版（推荐），如果你：
- 觉得李氏三拼的字体有点大
- 觉得英文键盘的字体有点小
- 想要开箱即用的最佳体验

### 选择全局字体版，如果你：
- 想完全自定义字体大小
- 要通过全局设置统一控制

### 选择 TX-02 专用版，如果你：
- 只用 TX-02 字体
- 不想配置全局字体

## 支持的键盘布局

- 26键拼音（横屏/竖屏，带反查版本）
- 9键拼音宫格（竖屏，带反查版本）
- 李氏双拼（lssp/lsspwsd）
- 英文26键
- 数字9键
- 符号键盘
- 表情键盘
- 浮动窗口（搜索、配置、切换）

## 主题特色

- 深色/浅色主题
- 自定义按键音效
- 原始布局设计
- 专业优化

## 构建

### 微调版：
```bash
python3 build_tuned.py
```

### 全局字体版：
```bash
python3 build.py
```

### TX-02 专用版：
```bash
python3 build_tx02.py
```
