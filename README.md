# 李氏三拼3x5a键盘主题

李氏三拼3x5a键盘主题包，保留原始字体，统一英文和数字字体大小。

## 下载

📦 [李氏三拼3x5a键盘_最终版.hskin](https://github.com/ovws/lishi-keyboard-theme/raw/main/李氏三拼3x5a键盘_最终版.hskin)

## 优化方案

### ✅ 解决的问题

1. **特殊符号显示正常** - 保留原始字体文件，粘贴/复制等符号正常显示
2. **英文字母大小一致** - 统一所有英文字母为 1.1em，不再大小不一
3. **数字清晰易读** - 统一数字大小为 1.1em
4. **中文不重叠** - 中文、符号、表情保持原始配置

### 📊 调整对比

| 布局类型 | 调整方案 | 效果 |
|---------|---------|------|
| 英文 (alphabetic) | 统一为 1.1em | 大小一致，清晰易读 |
| 数字 (numeric) | 统一为 1.1em | 数字输入舒适 |
| 中文 (pinyin) | 保持原样 | 避免重叠 |
| 符号 (symbolic) | 保持原样 | 特殊符号正常 |
| 表情 (emoji) | 保持原样 | 不需调整 |

### 🎯 特性

- ✅ **保留字体文件**：kafei.ttf 字体文件保留
- ✅ **统一英文大小**：所有英文字母 1.1em
- ✅ **统一数字大小**：所有数字 1.1em
- ✅ **中文不重叠**：保持原始大小
- ✅ **轻量级**：文件大小约 996KB
- ✅ **格式**：.hskin 格式

## 支持的键盘布局

- 26键拼音（横屏/竖屏，带反查版本）
- 9键拼音宫格（竖屏，带反查版本）
- 李氏双拼（lssp/lsspwsd）
- 英文26键 ⭐ 字体统一 1.1em
- 数字9键 ⭐ 字体统一 1.1em
- 符号键盘 ⭐ 特殊符号正常显示
- 表情键盘
- 浮动窗口（搜索、配置、切换）

## 主题特色

- 深色/浅色主题
- 自定义按键音效
- 优化的字体显示效果
- 统一的英文数字体验

## 构建

使用 `build.py` 脚本从原始主题包构建：

```bash
python3 build.py
```

脚本会自动：
1. 保留原始字体文件（特殊符号需要）
2. 统一英文字母大小为 1.1em
3. 统一数字大小为 1.1em
4. 保持中文、符号、表情原样
5. 打包为 .hskin 格式
