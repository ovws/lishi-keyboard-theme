# 李氏三拼3x5a键盘

李氏三拼3x5a键盘主题包。

## 版本说明

### 双字体版 (李氏三拼3x5a键盘_双字体版.zip)

- **中文**：HYHuaGuan_55W.ttf（汉仪华冠55W字体）
- **英文**：TX-02-Regular.otf（TX-02 Regular字体）
- **备用**：PingFangSC-Regular（苹方常规体）
- **文件大小**：约 5.2MB

### 无字体版 (李氏三拼3x5a键盘_无字体版.hskin)

- **特点**：不包含字体文件，使用系统默认字体
- **文件大小**：约 966KB
- **格式**：.hskin
- **优势**：体积小，加载快，适合自定义字体的用户

### 字体调整版 (李氏三拼3x5a键盘_字体调整版.hskin) ⭐ 最新

- **中文输入**：字体缩小至原来的 85%
- **英文输入**：字体放大至原来的 115%
- **文件大小**：约 997KB
- **格式**：.hskin
- **适用场景**：希望中英文字体大小有区分的用户

## 支持的键盘布局

- 26键拼音（横屏/竖屏，带反查版本）
- 9键拼音宫格（竖屏，带反查版本）
- 李氏双拼（lssp/lsspwsd）
- 英文26键
- 数字9键
- 符号键盘
- 表情键盘
- 浮动窗口（搜索、配置、切换）

## 主题特色

- 深色/浅色主题
- 自定义按键音效
- 优化的字体显示效果

## 安装说明

下载 `李氏三拼3x5a键盘_双字体版.zip` 并导入到你的输入法应用中。
