# 李氏三拼3x5a键盘主题

李氏三拼3x5a键盘主题包，提供两个版本。

## 下载

### 版本 1：全局字体版（推荐）

📦 [李氏三拼3x5a键盘_全局字体版.hskin](https://github.com/ovws/lishi-keyboard-theme/raw/main/李氏三拼3x5a键盘_全局字体版.hskin) (966KB)

**特点：**
- ✅ 无字体文件，无字体配置
- ✅ 使用系统默认字体或全局设置
- ✅ 体积最小
- ✅ 字体完全由你控制

### 版本 2：TX-02 专用版

📦 [李氏三拼3x5a键盘_TX02版.hskin](https://github.com/ovws/lishi-keyboard-theme/raw/main/李氏三拼3x5a键盘_TX02版.hskin) (2060KB)

**特点：**
- ✅ 内置 TX-02 完整字体系列（30个字体）
- ✅ 字体配置已正确设置
- ✅ 开箱即用 TX-02 字体
- ✅ 无需额外配置

## 版本对比

| 特性 | 全局字体版 | TX-02 专用版 |
|-----|-----------|------------|
| 字体文件 | 无 | TX-02 完整系列 (30个) |
| 字体配置 | 无 | fontFace 已配置 |
| 文件大小 | 966KB | 2060KB |
| 字体控制 | 全局设置 | 主题内置 |
| 推荐场景 | 想自由更换字体 | 只用 TX-02 |

## 使用建议

### 选择全局字体版，如果你：
- 想自由控制字体（TX-02、SF Pro、其他字体）
- 希望所有主题使用统一字体
- 想要最小体积

### 选择 TX-02 专用版，如果你：
- 只用 TX-02 字体
- 不想配置全局字体
- 希望开箱即用

## 支持的键盘布局

- 26键拼音（横屏/竖屏，带反查版本）
- 9键拼音宫格（竖屏，带反查版本）
- 李氏双拼（lssp/lsspwsd）
- 英文26键
- 数字9键
- 符号键盘
- 表情键盘
- 浮动窗口（搜索、配置、切换）

## 主题特色

- 深色/浅色主题
- 自定义按键音效
- 原始布局设计
- 专业优化

## 构建

### 全局字体版：
```bash
python3 build.py
```

### TX-02 专用版：
```bash
python3 /tmp/build_tx02.py
```
