# 李氏三拼3x5a键盘主题

李氏三拼3x5a键盘主题包，统一字体大小，无字体文件版本。

## 下载

📦 [李氏三拼3x5a键盘_统一字体版.hskin](https://github.com/ovws/lishi-keyboard-theme/raw/main/李氏三拼3x5a键盘_统一字体版.hskin)

## 特性

- ✅ **统一字体大小**：中文、英文、数字、符号等所有布局使用统一字体大小
- ✅ **无字体文件**：不包含任何字体文件，使用系统默认字体
- ✅ **轻量级**：文件大小约 965KB
- ✅ **格式**：.hskin 格式

## 支持的键盘布局

- 26键拼音（横屏/竖屏，带反查版本）
- 9键拼音宫格（竖屏，带反查版本）
- 李氏双拼（lssp/lsspwsd）
- 英文26键
- 数字9键
- 符号键盘
- 表情键盘
- 浮动窗口（搜索、配置、切换）

## 主题特色

- 深色/浅色主题
- 自定义按键音效
- 统一优化的字体显示效果

## 构建

使用 `build.py` 脚本从原始主题包构建：

```bash
python3 build.py
```
