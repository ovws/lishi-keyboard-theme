# 李氏三拼3x5a键盘主题

李氏三拼3x5a键盘主题包，使用 TX-02 字体，放大候选词显示。

## 下载

📦 [李氏三拼3x5a键盘_完美版.hskin](https://github.com/ovws/lishi-keyboard-theme/raw/main/李氏三拼3x5a键盘_完美版.hskin)

## v3 完美版优化方案

### ✅ 解决的问题

1. **英文字母使用 TX-02 字体** - 优雅的英文显示效果
2. **特殊符号正常显示** - 保留 kafei.ttf 用于中文和特殊符号
3. **候选词放大 30%** - 横排和展开候选词更清晰
4. **按键字母保持原样** - 不调整按键上的字母大小

### 📊 字体配置

| 用途 | 字体 | 说明 |
|-----|------|------|
| 英文字母 | TX-02-Regular | 专业英文字体 |
| 中文、符号 | kafei.ttf | 原始主题字体 |
| 候选词区域 | 继承上述配置 | 放大 30% |

### 🎯 特性

- ✅ **双字体方案**：TX-02 (英文) + kafei.ttf (中文/符号)
- ✅ **候选词放大**：横排、展开候选词字体放大 30%
- ✅ **按键原始大小**：按键上的字母保持原设计
- ✅ **特殊符号正常**：粘贴/复制等功能正常
- ✅ **文件大小**：约 1030KB
- ✅ **格式**：.hskin

### 📈 字体大小对比

**候选词区域（已放大 30%）：**
- 横排文字：1.19em → 1.547em
- 展开文字：1.125em → 1.4625em
- 横排序号：1.19em → 1.547em
- 展开序号：1.125em → 1.4625em

**按键区域（保持原样）：**
- 主键字符：保持原设计
- 上划字符：保持原设计

## 支持的键盘布局

- 26键拼音（横屏/竖屏，带反查版本）⭐ 候选词放大
- 9键拼音宫格（竖屏，带反查版本）⭐ 候选词放大
- 李氏双拼（lssp/lsspwsd）⭐ 候选词放大
- 英文26键 ⭐ TX-02 字体
- 数字9键
- 符号键盘 ⭐ 特殊符号正常
- 表情键盘
- 浮动窗口（搜索、配置、切换）

## 构建

使用 `build.py` 脚本从原始主题包构建：

```bash
python3 build.py
```

脚本会自动：
1. 添加 TX-02 字体文件
2. 保留 kafei.ttf 字体
3. 配置双字体方案
4. 放大候选词区域字体 30%
5. 保持按键字母原始大小
6. 打包为 .hskin 格式
