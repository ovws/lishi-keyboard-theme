# Lishi Keyboard Theme

自定义键盘主题字体文件

## 字体文件

### HYHuaGuan_65W_TX02_Mixed.ttf

这是一个混合字体，结合了：
- **中文部分**: 汉仪华冠 65W (HYHuaGuan 65W)
- **英文部分**: TX-02 Regular

该字体中的所有英文字符（包括基本 ASCII、Latin-1 补充和 Latin 扩展-A）都使用 TX-02 Regular 的字形，而中文字符保持原汉仪华冠 65W 的风格。

## 使用方式

1. 下载 `fonts/HYHuaGuan_65W_TX02_Mixed.ttf`
2. 安装到系统字体目录
3. 在键盘主题中配置使用该字体

## 字符范围

- **基本 ASCII** (U+0020 到 U+007E): 空格、标点、数字、大小写字母
- **Latin-1 补充** (U+00A0 到 U+00FF): 扩展拉丁字符
- **Latin 扩展-A** (U+0100 到 U+017F): 更多欧洲语言字符
- **中文字符**: 保留原字体字形
