# 李氏三拼3x5a键盘主题

李氏三拼3x5a键盘主题包，优化英文和数字字体大小，无字体文件版本。

## 下载

📦 [李氏三拼3x5a键盘_优化版.hskin](https://github.com/ovws/lishi-keyboard-theme/raw/main/李氏三拼3x5a键盘_优化版.hskin)

## 特性

- ✅ **英文字母放大 120%**：避免英文字母显示过小
- ✅ **数字放大 120%**：提升数字输入可读性
- ✅ **中文保持原样**：避免字符重叠，保持最佳显示效果
- ✅ **符号和表情不变**：保持原始设计的平衡感
- ✅ **无字体文件**：使用系统默认字体
- ✅ **轻量级**：文件大小约 966KB
- ✅ **格式**：.hskin 格式

## 调整说明

| 布局类型 | 调整比例 | 说明 |
|---------|---------|------|
| 英文 (alphabetic) | 120% | 字母显示更清晰 |
| 数字 (numeric) | 120% | 数字输入更舒适 |
| 中文 (pinyin) | 保持原样 | 避免重叠 |
| 符号 (symbolic) | 保持原样 | 维持平衡 |
| 表情 (emoji) | 保持原样 | 不需调整 |

## 支持的键盘布局

- 26键拼音（横屏/竖屏，带反查版本）
- 9键拼音宫格（竖屏，带反查版本）
- 李氏双拼（lssp/lsspwsd）
- 英文26键 ⭐ 字体放大
- 数字9键 ⭐ 字体放大
- 符号键盘
- 表情键盘
- 浮动窗口（搜索、配置、切换）

## 主题特色

- 深色/浅色主题
- 自定义按键音效
- 优化的字体显示效果
- 避免字符重叠

## 构建

使用 `build.py` 脚本从原始主题包构建：

```bash
python3 build.py
```

脚本会自动：
1. 删除所有字体文件
2. 清理字体配置
3. 只调整英文和数字字体大小
4. 保持中文、符号、表情原样
5. 打包为 .hskin 格式
