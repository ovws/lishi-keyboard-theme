# 李氏三拼3x5a键盘主题

李氏三拼3x5a键盘主题包，无字体配置，使用全局字体。

## 下载

📦 [李氏三拼3x5a键盘_全局字体版.hskin](https://github.com/ovws/lishi-keyboard-theme/raw/main/李氏三拼3x5a键盘_全局字体版.hskin)

## 设计理念

**完全删除字体配置，让你自由控制字体！**

- ✅ 删除所有字体文件
- ✅ 删除所有字体配置
- ✅ 使用系统默认字体
- ✅ 可通过输入法全局字体设置覆盖

## 特性

| 特性 | 说明 |
|-----|------|
| 字体文件 | 无 |
| 字体配置 | 无 |
| 字体来源 | 系统默认或全局设置 |
| 文件大小 | 约 966KB |
| 格式 | .hskin |

## 使用方法

1. 安装本主题包
2. 在输入法全局设置中配置你想要的字体（如 TX-02）
3. 全局字体设置会自动应用到本主题

## 支持的键盘布局

- 26键拼音（横屏/竖屏，带反查版本）
- 9键拼音宫格（竖屏，带反查版本）
- 李氏双拼（lssp/lsspwsd）
- 英文26键
- 数字9键
- 符号键盘
- 表情键盘
- 浮动窗口（搜索、配置、切换）

## 主题特色

- 深色/浅色主题
- 自定义按键音效
- 原始布局设计
- 轻量级体积
- 字体完全由你控制

## 构建

使用 `build.py` 脚本从原始主题包构建：

```bash
python3 build.py
```

脚本会自动：
1. 删除 fonts/ 目录
2. 删除 config.yaml 中的 fontFace 配置
3. 验证无字体配置残留
4. 打包为 .hskin 格式

## 为什么选择全局字体版？

✅ **完全自由**：你可以在输入法全局设置中使用任何字体  
✅ **体积更小**：不包含字体文件，更轻量  
✅ **统一管理**：所有主题使用同一全局字体配置  
✅ **无冲突**：不会出现字体配置不生效的问题
